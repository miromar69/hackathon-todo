# Claude Code Instructions – Phase I

This project is part of Hackathon II – Evolution of Todo.

For Phase I:

1. Read the spec file: `specs-history/phase1-task-crud-v1.md`.
2. Implement a Python 3 console application in `src/todo_app.py` that fully satisfies the spec.
3. Use clean, modular code with functions or a simple class.
4. The program must have a `main()` function and an entry point like:

   if __name__ == "__main__":
       main()

5. Use only the Python standard library.
6. Store tasks in memory only (do not use files or databases).
7. Provide helpful messages for invalid inputs.
